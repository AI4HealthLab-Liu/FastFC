Welcome to Fast Functional Connectivity Toolbox.

This toolbox is the result of an effort to improve efficiency for Functional Connectivity measures. You can download the executable files (right now only available for Windows x64 sytems) here. All functions are developed so that they can be called directly from Matlab's command line.

  • Include folder in Matlab's path.
